##  项目运行

#### 注意:由于项目已经npm run build编译打包完成 而且已经基于nodejs的一个express框架搭建好了一个server.js服务器 所以直接启动服务器 即可快速生成项目
```
git clone https://github.com/wensiyuanseven/SimpleBooks.git

cd SimpleBooks

npm install

node server.js

在地址栏中输入 localhost:9000
```
